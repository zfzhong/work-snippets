03/13
======
1. With codex we built a click-canvas web app that generates click-style diagrams. This is so cool. Why is this great? Using metapost to fine-grain control the layout and connection, but this requires a lot of coding effort. We decopule this: 1) using ui interface to wire blocks -- this creates the logic dependency of elements easily; 2) generate metapost code based on the dependency graph, and then realign these elements. So, the connecting part is easy using ui but difficult using code; the alignment part is easy using code but difficult to do manually through ui interface. What a design!
